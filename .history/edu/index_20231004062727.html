Next.js is a front-end framework powered by React, a free and open-source JavaScript library for capturing user interfaces of UI components. One of the most popular front-end frameworks, React has become a favorite among developers. The default deployments for React are modern stack websites and single page applications (SPAs). The former contains multiple pages like a traditional website, but provides pages implemented with a modern framework that provides users with an app-like experience by refreshing the current page instead of loading a new one from the server.

Motion graphics (sometimes mograph) are pieces of animation or digital footage which create the illusion of motion or rotation, and are usually combined with audio for use in multimedia projects. Motion graphics are usually displayed via electronic media technology, but may also be displayed via manual powered technology (e.g. thaumatrope, phenakistoscope, stroboscope, zoetrope, praxinoscope, flip book).
